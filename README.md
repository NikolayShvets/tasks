подготовка к собеседованиям
Задача 1 "Строки в стиле Си" (cstring)
Задача: на вход программе подается строка, содержащая слова, записанные латиницей, разделенные между собой различным
количеством других символов. Необходимо подсчитать количество слов в строке.
Ограничение: необходимо использовать строки в стиле Си
Вход: #$#@i   #$@#  ***love **(%$         you
Выход: 3
Задача 2 "Баланс скобок (курглые скобки)"
Задача: дана строка, содержащая скобки '(',')'. Необходимо проверить, что все открытые скобки закрыты,
а все закрытые скобки были открыты
Вход: (())()() Выход: True	Вход: ()(()(( Выход: False
